# 09. 把经验沉淀下来：Todo / 会话日志 / Memory

> CC-Panes 有三个"记事"工具，帮你把待办、工作记录、长期经验都留下来——而且它们都能被 AI 通过 [MCP](mcp-orchestration.md) 读写。

## Todo：待办管理

点左侧 ActivityBar 的 **TodoList** 图标进入全屏 Todo。

- **分作用域**：全局 / 工作空间 / 项目，还能区分外部（脚本生成）来源
- 优先级（高 / 中 / 低）、状态（待办 / 进行中 / 完成）
- **子任务**、标签、到期日期、提醒、循环重复
- 拖拽排序、按 标签 / 状态 / 优先级 / 作用域 分组、「我的一天」视图

## 会话日志（Journal）：记录每次工作

右键**工作空间** → 打开会话日志。它按工作空间记录一条条会话总结：**标题、总结、涉及的 Git 提交**。适合回顾“这个工作空间最近都干了啥”。

## Memory：长期经验与决策

右键**项目** → 打开 Memory Manager。把值得长期记住的东西存下来，AI 之后能召回作为上下文：

- **分类**：决策 / 经验教训 / 偏好 / 模式 / 事实 / 计划
- **重要度**：1–5 星（越高越优先被召回）
- **作用域**：全局 / 工作空间 / 项目 / 会话
- 按 作用域 / 分类 / 重要度 / 标签 搜索过滤，增删改

> 这三样都能被 AI 通过 MCP 读写，所以“记下来的东西”不只是给你看，也会进入 AI 的工作上下文——你说“记住以后都这样”，它就能写进 Memory。
