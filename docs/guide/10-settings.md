# 10. 设置详解

> 点左侧底部的 **设置**（齿轮）图标，或按 `Ctrl+,` 打开。设置面板左侧分成下面这些区。

## 通用

- 关闭窗口时最小化到托盘、开机自启
- 语言（简体中文 / English）
- 默认 CLI 工具（自我对话、会话恢复等场景使用）
- 搜索范围（仅工作空间 / 全盘——全盘更耗内存、首次索引慢）
- 数据目录（查看大小、迁移到别处、恢复默认；改后需重启）

## 通知

启用通知，以及只在 任务退出时 / 等待输入时 / 窗口失焦时 才通知。

## Provider

管理 AI Provider（连哪条模型通道）。可从预设创建或手动配：Anthropic、Bedrock、Vertex、OpenAI、Gemini、Kimi、GLM、OpenCode、Cursor、自定义代理、配置文件 等类型，填 API Key / Base URL。给工作空间绑定后，启动终端时会自动注入对应环境变量。（概念见 [03. 核心概念](03-core-concepts.md)）

## 代理

HTTP / SOCKS5 代理（注意：SOCKS5 不被所有工具支持，推荐 HTTP），填主机 / 端口 / 用户名 / 密码 / 排除列表，可测试连接。

## 终端

字号、字体、光标样式（方块 / 下划线 / 竖线）、光标闪烁、终端主题（跟随应用 / 深色 / 浅色）、回滚行数、渲染器（自动优先 WebGL / 尝试 WebGL / DOM 降级）。

## 语音输入

每个终端窗格可录音，识别后把文字写进终端（不自动提交）。可选语音服务（DashScope / Qwen-ASR、小米 MiMo）、填 API Key、选语种（中 / 粤 / 英 / 日 / 韩 / 自动）、模型、最长录音时长等。

## cc酱

桌面宠物 cc酱 的动画与交互配置。

## 快捷键

查看并自定义所有快捷键：点一项、按下新组合键即可重绑，有冲突会提示。（默认值见 [05. 终端与分屏](05-terminal-and-panes.md) 的速查表）

## Shared MCP

管理共享的 MCP Server（启动 / 停止、查看状态和端口），供各实例和 Skill 调用。

## 截图（仅 Windows / Linux）

截屏一块区域并把文件路径复制到剪贴板，方便贴给 Claude Code 引用图片。可设快捷键和保留天数。

## 关于

应用名、版本、技术栈，检查更新，打开日志目录。
