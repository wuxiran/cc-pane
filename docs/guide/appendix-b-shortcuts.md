# 附录 B：快捷键速查

> 以下是默认快捷键，都可以在 **设置 → 快捷键** 里自定义重绑。

## 窗口与视图

| 快捷键 | 动作 |
| --- | --- |
| `Ctrl+B` | 折叠 / 展开侧边栏 |
| `F11` | 切换全屏 |
| `Ctrl+M` | 切换迷你模式 |
| `Ctrl+,` | 打开设置 |

## 标签页

| 快捷键 | 动作 |
| --- | --- |
| `Ctrl+T` | 新建标签 |
| `Ctrl+W` | 关闭标签 |
| `Ctrl+Tab` | 下一个标签 |
| `Ctrl+Shift+Tab` | 上一个标签 |
| `Ctrl+1` ~ `Ctrl+9` | 切换到第 1~9 个标签 |

## 分屏

| 快捷键 | 动作 |
| --- | --- |
| `Ctrl+\` | 向右分屏 |
| `Ctrl+-` | 向下分屏 |
| `Alt+←/→/↑/↓` | 切换到左 / 右 / 上 / 下面板 |

## 其它

| 快捷键 | 动作 |
| --- | --- |
| `Ctrl+Alt+M` | 语音输入 |
| `Ctrl+Shift+S` | 截图（开发版为 `Ctrl+Alt+Shift+S`） |

## 一个常见疑问

当焦点在终端里时，少数与终端 / CLI 冲突的快捷键（`Ctrl+B`、`Ctrl+T`、`Ctrl+W`、`Ctrl+M`、`Ctrl+\`、`Ctrl+-`）会**放行给终端**，让 Claude Code 等程序自己处理。想用这些 App 级动作时，先把焦点点到终端外，或改用右键菜单。
