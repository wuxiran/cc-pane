# 08. Local History：文件版本回滚与对比

> 不靠 Git 也能找回文件的历史版本。CC-Panes 会**自动捕获**项目里文件的改动，存成一条条本地快照。

## 它能干什么

- 文件改坏了 / 误删了，**找回任意历史版本**
- 对比两个版本之间的差异
- 按分支、按标签筛选历史

## 怎么打开

- 右键编辑器里的文件 → 「文件历史」
- 或左侧边栏 工具 → 「文件历史」

## 界面与操作

两步式：先选一个**文件**，再看它的**版本列表**（最新在上）。对某个版本可以：

- **看差异（Diff）**：和别的版本或当前内容比
- **看完整内容**
- **恢复**到这个版本
- 还能找回**已删除的文件**

## 标签：快速定位关键节点

历史里会自动打标签，用颜色区分来源：

- 🟡 Git 提交　🟣 Claude 会话　🔵 你手动加的　🟢 构建　🔴 恢复操作

可以按标签或按分支筛选，快速跳到“重构前”“上次提交时”这种关键点。它是**分支感知**的——切了分支，看到的历史也跟着该分支走。
