# 06. 文件浏览与编辑、Markdown 预览

> 不用切回 IDE，CC-Panes 内置了文件浏览器和编辑器，看代码、改文件、预览 Markdown 都能在这里完成。

## 打开文件浏览器

点左侧 ActivityBar 的 **文件**（Files 图标）切到文件浏览视图。顶部有一排按钮：后退 / 前进 / 上一级 / 刷新 / 折叠全部 / 回到项目根目录 / 新建文件 / 新建文件夹。

## 文件树右键菜单

在文件或文件夹上右键（选项随文件 / 文件夹略有不同）：

- 在编辑器中打开（文件）／在终端中打开（文件夹）
- 在系统文件管理器中打开 ／ 在系统文件管理器中显示
- 复制绝对路径 ／ 复制相对路径
- 新建文件 ／ 新建文件夹（文件夹）
- 重命名 ／ 复制到… ／ 移动到… ／ 删除

## 编辑器

内置 **Monaco 编辑器**（VS Code 同款内核），按扩展名自动识别语言高亮（TS/JS、JSON、Python、Go、Rust、Java、C/C++、Shell、SQL 等一长串）。单文件上限 5MB。

针对几类文件有特殊查看：

- **Markdown**：支持 编辑 / 预览 / 分屏 三种模式，预览是 GitHub 风格（react-markdown + GFM）
- **图片**：PNG / JPG / GIF / WebP / BMP / ICO 等，可缩放（放大 / 缩小 / 适应窗口 / 1:1），并显示分辨率和文件大小
- **JSON**：专门的 JSON 编辑器，带格式校验和高亮

## 快速打开最近文件

有一个「最近文件」快速选择器：弹出后能搜索文件名，在「打开中的标签」和「最近」之间用上下键 + 回车快速跳转，省得在文件树里翻。
