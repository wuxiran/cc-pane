# 附录 C：常见问题（FAQ）

## 一个工作空间下放了多个关联项目（比如前后端），开发时开一个会话框还是多个？

开**一个**就够了。把前后端这类关联项目放进同一个工作空间后，在一个会话框（终端会话）里就能同时推进它们——不必为每个项目单独开会话框。

## 会话框能读取工作空间下所有项目的文件吗？项目分布在不同磁盘也行吗？

能。工作空间会用 `.ccpanes/projects.csv` 记录它下面所有项目的信息（包括分布在不同磁盘上的项目路径），所以会话框能访问到工作空间下各个项目的文件，不受项目在哪个磁盘的限制。

## 同一个工作空间下的多个会话框之间，能互相读取吗？

可以。同一工作空间下的会话框之间是互通的，能互相读取。
