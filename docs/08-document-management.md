# 阶段 8：文件浏览与 Markdown 预览

## 目标

在应用内浏览工作空间文件和预览 Markdown 文档。

## 状态

📋 待实现

## 背景说明

用户需要在 CC-Panes 中查看项目文件结构和预览 Markdown 文档，而不需要切换到其他编辑器。本阶段只做只读预览，不实现编辑功能。

## 核心功能

### 1. 工作空间文件浏览器

树形显示项目文件目录，支持展开/折叠、文件类型图标。

### 2. Markdown 预览

渲染 Markdown 为 HTML 预览，支持 GFM（表格、任务列表、代码块等）。

### 3. 文件快速查看

支持常见代码文件的语法高亮查看。
